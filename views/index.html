Exercise tracker

GET users's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Enter your GET query into the url in the following way:

/api/exercise/log?userId=5faaeef1a37a3f00cf1c3500&from=2001-01-01&to=2002-01-01&limit=3

from, get and to are optional parameters, just enter the userId for all records for that user.
